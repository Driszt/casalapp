# ❤️ CasalApp

O vosso espaço privado de casal.

## Como fazer o deploy no Vercel

### 1. Cria uma conta no GitHub (se não tiveres)
- Vai a github.com e regista-te

### 2. Cria um repositório novo
- Clica em "New repository"
- Chama-lhe "casalapp"
- Clica "Create repository"

### 3. Faz upload dos ficheiros
- Arrasta toda a pasta casalapp para o repositório

### 4. Deploy no Vercel
- Vai a vercel.com
- Clica "Sign up with GitHub"
- Clica "Add New Project"
- Escolhe o repositório "casalapp"
- Clica "Deploy"
- Em ~2 minutos tens o link! 🎉

## Funcionalidades
- 💬 Chat em tempo real entre os dois
- 🤖 Bot offline inteligente
- ✅ Tarefas partilhadas
- 📅 Calendário com eventos
- 👤 Estado de disponibilidade
- 🔐 Login com email privado
